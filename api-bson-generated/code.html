A class representation of the BSON DBRef type.

class DBRef()¶
Arguments:
  • namespace (string) – the collection name.
  • oid (objectid) – the reference ObjectID.
  • [db] (string) – optional db name, if omitted the reference is local to the current db.
Returns:

dbref
